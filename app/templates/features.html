Image Sorting

Medical images exported from an imaging system or PACS system are often saved with filenames based upon unique identifiers for the images. The primary goal of DICOMsort is to reorganize and rename these images in a more user-friendly way. The software currently supports the following:

  • Multi-level sorting based upon DICOM header
  • Custom configuration of default sorting options
  • Recursive token-based sorting
  • Custom File Naming

Anonymization

DICOM anonymization is essential in many applications to protect patients' privacy. In addition to sorting, this software is able to perform anonymization:

  • Anonymizes customizable fields
  • Allows user to specify replacement values
  • Replacement values can depend on other values
  • In-place anonymization
  • Simulatneous sorting and anonymization
